# Vyper Team - Official Website

Sitio web oficial de Vyper Team, una comunidad gaming competitiva enfocada en el crecimiento, disciplina y trabajo en equipo.

## Juegos Activos

- Valorant
- Honor of Kings
- BloodStrike
- Mobile Legends
- Free Fire

## Tecnologías

- Next.js 16
- React 19
- Tailwind CSS v4
- TypeScript
- Framer Motion
- shadcn/ui

## Instalación Local

\`\`\`bash
# Instalar dependencias
npm install

# Ejecutar en modo desarrollo
npm run dev

# Compilar para producción
npm run build

# Ejecutar en producción
npm start
\`\`\`

## Deployment

Este proyecto está optimizado para deployment en Vercel o Netlify.

### Vercel (Recomendado)
1. Conecta tu repositorio de GitHub
2. Vercel detectará automáticamente Next.js
3. Deploy automático

### Netlify
1. Build command: `npm run build`
2. Publish directory: `.next`
3. Deploy automático desde GitHub
